### 6. Знайти кількість значень < 0 для кожного стовпця. Використати свою функцію.
```R
my_func <- function(x){
    ret_arr <-NULL
    for (i in 1:ncol(x)){
        ret_arr[i]<-sum(x[, i] < 0)
    }
    ret_arr
}

my_func(mat)
```
```cmd
> my_func(mat)
[1] 6 4 3 5 2
```
